## Project Description

Retail Inventory and Billing Management System is a console-based Java OOP project designed for small retail shops and stores. It helps in managing products, customer orders, billing, and stock records in a simple and organized way.

---

## Purpose

The main purpose of this project is to reduce manual work in retail shops by automating inventory management and billing processes. It helps shop owners manage products, calculate bills, store orders, and track low stock items easily.

---

## Main Modules

* Admin Login System
* Product Management
* Order Management
* Billing System
* Low Stock Alert
* Order History

---

* Encapsulation
* Inheritance

* Collections (ArrayList)
* Exception Handling
* JDBC Database Connectivity

---

## How to Run

1. Open project folder in terminal
2. Compile all Java files using:
javac -cp ".;mysql-connector-j-9.7.0.jar" *.java

3. Run the project using:
java -cp ".;mysql-connector-j-9.7.0.jar" Main

4. Make sure MySQL is running before database connection
